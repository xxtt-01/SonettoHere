# 记忆管理工具集

提供对 AI 助手长期记忆的完整 CRUD 操作。记忆以 YAML 文件形式持久化存储在 `config/personas/memory.yaml` 中。

## 分区（section）

每条记忆属于一个主题分区，例如：
- `身份` — 用户的基本身份信息（教育、职业、家乡等）
- `音乐` — 虚拟歌手、声库、歌曲、专辑、创作者
- `品味` — 电影、美食、UP主、品牌偏好等
- `地点与路径` — 具体地点和文件系统路径
- `瞬间` — 即时观察和感受（天气、正在做的事、念头）
- `时效待办` — 有截止日期的事项（作业、预约、考试）

可以创建新的分区（1-4字中文名词）来组织记忆。

## 可用工具

| 工具 | 说明 |
|------|------|
| `list_memories` | 查看所有记忆条目概览（带 ID 和分区，每条描述已截断以节省上下文） |
| `read_memories` | 根据 ID 读取单条记忆的完整内容（含变更历史） |
| `create_memory` | 添加一条新记忆 |
| `update_memory` | 修改已有记忆 | 
| `delete_memory` | 删除一条记忆 |
| `merge_memories` | 合并两条相似记忆 |

## 使用原则

1. 先用 `list_memories` 获取所有条目的 ID 和概览
2. 每条记忆只包含一个独立事实
3. 使用第三人称描述
4. 不使用相对时间词汇（今天、明天等），使用绝对日期
